birds-HC.ipynb is heuristic hard coding combined with Boids flocking function code result, which is the first stage content in the report.

birds-HC-A*.ipynb is the second stage of the report that adds the A* algorithm.

birds_HC_Use-model.ipynb is the part of the training model in the third stage. The model has been trained as bird_model_ppo.zip, and retraining will take a lot of time.

birds-HC-Try-model.ipynb is the code to use the model.
